# Agent 设计模式之美

**极客时间 · 黄佳**

---

## 课程简介

这是一门关于 LLM Agent 设计的工程课。不是理论课，是从真实代码库里提炼设计模式的实战课。

核心框架：**双轴矩阵** — 7 种认知功能（感知、记忆、推理、规划、行动、协作、治理）× 6 种执行拓扑（链式、层级、循环、星形、总线、并行）= 42 格，28 个已验证模式。
